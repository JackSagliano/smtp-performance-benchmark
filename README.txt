smtp4dev permette di creare un SMTP server in locale.

Runnare da shell:
            winget install smtp4dev
E poi:
             Rnwood.Smtp4dev
Successivamente, una volta installato smtp4dev e averlo lanciato, recarsi sulle impostazioni
(ingranaggio in alto a destra) e recarsi prima nella sezione "SMTP Server" e togliere la spunta
da "Allow Any Credentials" e poi recarsi nella sezione "Users" e aggiungere l'utente o gli utenti
che il server accetta quando si effettua l'autenticazione tramite il form HTML lato client
